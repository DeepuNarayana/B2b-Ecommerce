# B2B Order Management System

A FastAPI-based order management system for B2B e-commerce platform.

## Setup

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

2. Run the application:
   ```bash
   uvicorn app.main:app --reload
   ```

## Exercises

This project implements the exercises from the technical assessment:

- Exercise 1: Domain Modeling & Business Rules
- Exercise 2: API Design & Error Handling
- Exercise 3: Architecture & Code Review
